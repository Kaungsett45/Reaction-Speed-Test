# Reaction-Speed-Test

## Prerequisites
- Node.js (v14 or higher)
- npm

## Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd Reaction-Speed-Test
```

2. Install backend dependencies:
```bash
cd backend
npm install
```

3. Install frontend dependencies:
```bash
cd frontend/speedtest
npm install
```

## Dependencies

### Backend
- express: ^5.1.0
- cors: ^2.8.5
- nodemon: ^3.1.10

### Frontend
- react: ^19.1.1
- react-dom: ^19.1.1
- vite: ^7.1.7
- tailwindcss: ^4.1.16
- @shadcn/ui: ^0.0.4
- axios: ^1.13.1

## Running the Application

1. Start the backend:
```bash
cd backend
npm start
```

2. Start the frontend (in a new terminal):
```bash
cd frontend/speedtest
npm run dev
```
